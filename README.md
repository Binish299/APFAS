# Nepali-English Speech Trainer

This local workspace contains the codebase for the offline, voice-based speech coach tailored for Nepali speakers learning English.

## Structure
- `backend/`: FastAPI API, Librosa extraction, and local Whisper modules.
- `frontend/`: React components, custom Web Audio API recording module, and dashboard.
- `database/`: SQLite backend and abstract repository layer.
